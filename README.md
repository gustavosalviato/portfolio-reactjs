## Guia de instalação

1- Clone o repositório
```sh
git clone https://github.com/gustavosalviato/portfolio-reactjs.git
```

2- Instalar as depedências
```sh
npm install
```

2- Por fim, rode o projeto
```sh
npm run dev
```
